# Voice Leading UX (starter)

## Goal
Make voice leading feel like a single switch that instantly improves musical flow.

## Primary control
- Bank-wide dropdown: OFF / Smooth / Cinematic / Neo-Soul / etc.

## Secondary controls (collapsed by default)
- Preserve bass
- Preserve top note
- Max jump
- Reset rules (inactivity/manual)

## Feedback
- subtle motion lines
- common-tone highlight
