# Style Guide (starter)

## Brand attributes
- Bold
- Performance-driven
- Premium
- Minimal but powerful

## Visual direction
- Dark UI, high contrast
- Tactile chord blocks
- Clear hierarchy, strong grid
- Subtle glow accents (used sparingly)

## Typography
- Use a modern sans for UI
- Use consistent weights for hierarchy

## Interaction
- Everything should feel playable (fast hover/pressed states)
- Avoid deep menus; prefer popovers and contextual panels
