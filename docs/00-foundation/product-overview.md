## Key Principles

1. **Jobs-First Thinking** â€“ Features are evaluated against the 10 core jobs we're solving, not by feature count
2. **Phased Development** â€“ We intentionally defer complex features (realtime playback) to later phases
3. **Persona Diversity** â€“ Different personas want different things; we design for the primary persona first but don't ignore secondary personas
4. **Voice Leading Focus** â€“ The core innovation is sophisticated voice leading; everything else is enabler
