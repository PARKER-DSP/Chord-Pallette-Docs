## Key Principles

1. **Jobs-First Thinking** ‚Äì Features are evaluated against the 10 core jobs we're solving, not by feature count
2. **Phased Development** ‚Äì We intentionally defer complex features (realtime playback) to later phases
3. **Persona Diversity** ‚Äì Different personas want different things; we design for the primary persona first but don't ignore secondary personas
4. **Voice Leading Focus** ‚Äì The core innovation is sophisticated voice leading; everything else is enabler
5. **Learning Through Doing** ‚Äì Customers should understand harmonic principles while using the product
